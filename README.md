# DETECCIÓN DE PHISHING DE SITIOS WEB MEDIANTE TÉCNICAS DE MACHINE LEARNING Y ANÁLISIS DINÁMICO DE URLS 

Este repositorio contiene el código desarrollado para mi Trabajo de Fin de Grado (TFG).

## Contenido
- Notebook principal realizado en Google Colab (.ipynb)
- Script equivalente en Python (.py)

## Objetivo
El objetivo de este proyecto es desarrollar y evaluar un sistema de detección de sitios web de phishing utilizando técnicas de machine learning y características asociadas a URLs y dominios.

Este enfoque combina:
- Características léxicas extraídas de las URLs
- Características del dominio e infraestructura
- Modelos de aprendizaje supervisado

## Modelos utilizados
- Regresión Logística (modelo baseline)
- Random Forest (modelo principal)

## Dataset
El dataset utilizado en este proyecto ha sido obtenido de Zenodo e incluye URLs que son legítimas y de phishing
